# Dotty Homes — dottyhomes.com

Marketing site for Dotty Homes, a residential construction & remodeling
company serving Los Angeles and Orange County.

Static site — plain HTML, CSS, and vanilla JavaScript. No build step.

- `index.html` — single-page site: hero slideshow, work photo wall, about, contact
- `styles.css` — section styles
- `app.js` — hero slideshow + scroll-reveal
- `photos/` — imagery (`hero-*` slideshow, `home-*` work wall, `floorplan.png`)

## Deploy

Hosted on GitHub Pages, served at **dottyhomes.com** (custom domain set via `CNAME`).
Pushing to `main` redeploys automatically.
